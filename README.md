# 基于 RR 间期时序特征的房颤智能诊断模型

本项目是机器学习课程实验，旨在仅利用心电图中的 RR 间期序列，构建一个高准确率的房颤（AF）/非房颤（Non-AF）二分类模型。

## 📌 研究流程
1.  **数据探索**：通过均值、方差等统计量，发现 AF 组具有显著更高的心率变异性（方差均值 1277 vs 471）。
2.  **特征工程**：构建包含基础统计量（均值、方差）和高阶统计量（偏度、峰度）的多维特征集。
3.  **模型对比**：系统性地训练并评估了 K近邻（KNN）和朴素贝叶斯（NB）分类器。
4.  **模型优化**：应用主成分分析（PCA）进行降维，探究特征数量对 KNN 性能的影响。

## 🛠 技术栈
- Python, Scikit-learn, Pandas, Matplotlib

## 💡 亮点
- **医学洞见**：特征选择直接源于房颤的病理生理机制。
- **方法论完整**：覆盖了从数据探索到模型评估的完整 ML pipeline。
- **批判性思维**：深入分析了不同算法的优劣及适用场景。
